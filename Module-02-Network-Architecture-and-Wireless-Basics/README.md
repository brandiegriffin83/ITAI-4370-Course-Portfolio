# Module 2: Network Architecture and Wireless Basics

## Module Overview

Module 2 expanded my understanding of modern wireless communication networks by introducing network architecture, wireless technologies, and the transition from 4G LTE to 5G. I explored how core network components work together to provide fast, reliable, and scalable communication services while learning how Artificial Intelligence improves resource management and network performance.

---

# Learning Objectives

By completing this module, I learned to:

- Explain the differences between 4G EPC and the 5G Core.
- Understand Service-Based Architecture (SBA).
- Describe Network Slicing.
- Explain Multi-access Edge Computing (MEC).
- Understand the role of Artificial Intelligence in resource allocation.
- Explain the purpose of the Network Repository Function (NRF).
