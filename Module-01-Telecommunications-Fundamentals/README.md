# Module 1: Telecommunications Fundamentals

## Module Overview

Module 1 introduced the fundamental concepts of telecommunications and communication systems. I learned how information is transmitted between devices using transmitters, communication channels, and receivers. This module also introduced the differences between analog and digital communications, communication system performance metrics, and the basic signal flow process used in modern wireless networks.

---

# Learning Objectives

By completing this module, I learned to:

- Explain the purpose of telecommunications systems.
- Identify the components of a communication system.
- Compare analog and digital communication.
- Describe different communication media.
- Understand bandwidth, latency, and noise.
- Analyze signal flow using Python.
