# Module 2: Network Architecture and Wireless Basics

## Overview

Module 2 expanded my understanding of wireless communication systems by introducing modern network architecture and the technologies that support 4G LTE and 5G communications. Through the assignment and laboratory exercise, I explored the differences between the 4G Evolved Packet Core (EPC) and the 5G Core (5GC), the importance of network slicing, Multi-access Edge Computing (MEC), Artificial Intelligence in network resource management, and radio frequency (RF) propagation modeling using Python.

# Laboratory 02

## RF Propagation Modeling with Python

### Description

In this laboratory, I used Python to model Free-Space Path Loss (FSPL) and analyze how wireless signals weaken as transmission distance increases. Using mathematical calculations and data visualization, I simulated RF propagation at 2.4 GHz and interpreted the resulting signal attenuation graph.

### Skills Demonstrated

- Python Programming
- RF Propagation Modeling
- Data Visualization
- Mathematical Modeling
- Wireless Communications

# What I Learned

This module strengthened my understanding of how modern wireless communication networks operate. I learned why the 5G Core provides greater flexibility than previous network generations and how technologies such as Network Slicing and Multi-access Edge Computing improve performance for different applications. The RF propagation laboratory reinforced these concepts by allowing me to model signal loss using Python and visualize how distance affects wireless communication.

---

# Key Takeaways

- The 5G Core is more flexible and scalable than the 4G EPC.
- Network Slicing allows one physical network to support multiple virtual networks.
- MEC reduces latency by processing data closer to users.
- Artificial Intelligence improves network resource management.
- RF signal strength decreases as transmission distance increases.
