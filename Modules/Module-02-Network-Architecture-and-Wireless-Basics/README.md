# Laboratory 02: RF Propagation Modeling with Python

## Problem Statement

The purpose of this laboratory was to model wireless signal propagation using the Free-Space Path Loss (FSPL) equation. The exercise demonstrated how transmission distance affects signal strength and provided a visual representation of RF signal attenuation using Python.

## Methods and Tools Used

- Google Colab
- Python
- NumPy
- Matplotlib
- Free-Space Path Loss (FSPL) Formula

## Description

In this laboratory, I used Python to model Free-Space Path Loss (FSPL) and analyze how wireless signals weaken as transmission distance increases. Using mathematical calculations and data visualization, I simulated RF propagation at 2.4 GHz and interpreted the resulting signal attenuation graph.

## Results

The Python simulation successfully modeled Free-Space Path Loss across multiple transmission distances. The resulting graph showed that signal attenuation increases as distance from the transmitter grows, demonstrating one of the fundamental principles of wireless communication. This laboratory helped reinforce how mathematical modeling and visualization can be used to analyze RF propagation in telecommunications networks.
