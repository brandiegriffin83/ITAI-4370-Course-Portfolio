# Laboratory 03: Network Traffic Prediction Using Supervised Learning

## Problem Statement

The purpose of this laboratory was to develop a supervised machine learning model capable of predicting future telecommunications network traffic using historical data. The project demonstrated how predictive analytics can improve network planning and resource allocation.

## Methods and Tools Used

- Google Colab
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Random Forest Regression

## Description

In this laboratory exercise, I developed a supervised machine learning model using Python to predict network traffic patterns. A Random Forest Regression model was trained using historical telecommunications data to forecast future network traffic. Model performance was evaluated using Mean Squared Error (MSE), R² scores, feature importance analysis, and prediction visualizations.

## Results

The machine learning model successfully predicted future network traffic based on historical telecommunications data. The prediction graphs demonstrated that the Random Forest Regression model closely matched the actual traffic patterns, while feature importance analysis identified the variables that contributed most to the model's predictions. This laboratory illustrated how Artificial Intelligence can improve network planning and optimize resource allocation.
