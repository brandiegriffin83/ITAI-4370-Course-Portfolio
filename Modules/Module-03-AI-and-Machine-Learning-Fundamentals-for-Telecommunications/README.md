# Assignment 03

## Artificial Intelligence and Machine Learning in Telecommunications

### Description

This assignment explored how Artificial Intelligence and Machine Learning improve modern telecommunications systems. Topics included AI-assisted radio resource management, Self-Organizing Networks (SON), predictive maintenance, Open RAN architecture, and the functions of the Near-Real-Time and Non-Real-Time RAN Intelligent Controllers (RICs).

### Skills Demonstrated

- Artificial Intelligence
- Machine Learning
- Open RAN
- Self-Organizing Networks (SON)
- Predictive Maintenance
- Critical Thinking

# Laboratory 03

## Network Traffic Prediction Using Supervised Learning

### Description

In this laboratory exercise, I developed a supervised machine learning model using Python to predict network traffic patterns. A Random Forest Regression model was trained using historical telecommunications data to forecast future network traffic. I evaluated model performance using Mean Squared Error (MSE), R² scores, feature importance analysis, and prediction visualizations.

### Skills Demonstrated

- Python Programming
- Machine Learning
- Random Forest Regression
- Data Analysis
- Feature Engineering
- Predictive Analytics
- Data Visualization
