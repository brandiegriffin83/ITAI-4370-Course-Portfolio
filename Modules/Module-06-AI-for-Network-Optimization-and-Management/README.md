# Laboratory 04: AI-Based Network Traffic Prediction

## Problem Statement

The objective of this laboratory was to analyze telecommunications network traffic using Artificial Intelligence and machine learning techniques. Historical traffic data was used to develop predictive models capable of forecasting future network demand and supporting intelligent network optimization.

## Methods and Tools Used

- Google Colab
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- TensorFlow/Keras
- ARIMA
- Linear Regression
- LSTM Neural Network

---

## Description

During this laboratory exercise, I developed multiple machine learning models to predict future network traffic using historical telecommunications data. The project included data preprocessing, feature engineering, seasonal decomposition, model training, performance evaluation, and visualization of prediction results. Multiple prediction models were compared to determine which produced the most accurate forecasts.

---

## Results

The predictive models successfully analyzed historical network traffic and generated future traffic forecasts. Linear Regression, ARIMA, and LSTM models were evaluated using performance metrics including Mean Squared Error (MSE), Mean Absolute Error (MAE), and Root Mean Squared Error (RMSE). Comparing multiple algorithms demonstrated how Artificial Intelligence can improve traffic prediction accuracy and support proactive network optimization.
