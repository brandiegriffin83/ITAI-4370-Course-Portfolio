# Laboratory 01: Communication System Modeling

## Problem Statement

The purpose of this laboratory was to model a basic communication system and simulate digital signal transmission using Python. The exercise demonstrated how Binary Phase Shift Keying (BPSK) modulation allows digital information to be transmitted and recovered while observing the effects of noise on communication signals.

## Methods and Tools Used

- Google Colab
- Python
- NumPy
- Matplotlib
- Binary Phase Shift Keying (BPSK) Simulation

## Description

During this laboratory exercise, I created a communication system model and explored how data travels from a transmitter to a receiver through a communication channel. Using Python, I simulated Binary Phase Shift Keying (BPSK) signal transmission to better understand signal modulation and the effects of noise during communication.

## Results

The Python simulation successfully demonstrated the transmission and recovery of digital signals using Binary Phase Shift Keying. The generated graphs illustrated how noise affects communication signals and how digital modulation techniques improve signal reliability. This exercise reinforced the importance of signal processing in modern telecommunications systems.
