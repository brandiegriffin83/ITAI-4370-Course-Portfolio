# Module 1: Telecommunications Fundamentals

## Overview

Module 1 introduced the core concepts of telecommunications and provided the foundation for understanding how information is transmitted across communication systems. Through the assignment and laboratory activities, I learned about the components of communication systems, analog and digital communication, network topology, and the basic principles of signal transmission. This module established the knowledge needed for the more advanced topics covered later in the course.

---

# Learning Objectives

After completing this module, I was able to:

- Define a telecommunications system and explain its purpose.
- Identify the major components of a communication system.
- Differentiate between analog and digital communication.
- Explain voice, data, video, and multimedia communications.
- Describe how network topology affects communication.
- Understand the relationship between bandwidth, latency, and noise.

# Laboratory 01

## Communication System Modeling

### Description

During this laboratory exercise, I created a communication system model and explored how data travels from a transmitter to a receiver through a communication channel. Using Python, I simulated Binary Phase Shift Keying (BPSK) signal transmission to better understand signal modulation and the effects of noise during communication.

### Skills Demonstrated

- Python Programming
- Signal Processing
- Communication Modeling
- BPSK Modulation
- Data Visualization

# Key Takeaways

- Telecommunications systems provide the foundation for all modern communication technologies.
- Digital communication is generally more reliable than analog communication because it is less susceptible to noise.
- Communication quality depends on factors such as bandwidth, latency, and interference.
- Python can be used to model communication systems and visualize signal transmission.
- This module established the technical foundation for the remainder of the course.
